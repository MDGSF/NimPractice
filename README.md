# NimPractice

## 编译运行

```
nim -h
nim --fullhelp
nim c helloworld.nim
nim c -r helloworld.nim
nim c -r --verbosity:0 helloworld.nim
nim c -d:release helloworld.nim
nim --cc:clang -d:release c hello // compile with Clang instead of GCC
```
